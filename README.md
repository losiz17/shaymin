## About shaymin
* bot on slack

## 使用言語

* ruby 2.4.2

## setting

1.slackでbotを作成してAPI Tokenを取得する

2.取得したTokenを環境変数として保存
`$export SLACK_API_TOKEN=ここに入れる`

3.docomo developerで雑談対話のapi keyを取得する

4.取得したapi keyを環境変数として保存
`$export APIKEY=ここに入れる`
